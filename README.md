Tic-Tac-Toe

Schulprojekt um das programmieren in C++ zu lernen.

Spielablauf:
Enter drücken
1 oder 2 für Einzel- oder Mehrspielermodus
Spielen bis drei gleiche Zeichen horizontal, vertikal oder diagonal zueinander sind oder das Spielfeld voll ist
1 oder 2 für nochmal spielen oder beenden

Verwendet habe ich;
C++
QT Creator

Das Spiel startet man, indem man die CMakeLists und den Code in QT Creator öffnet und auf den Run-Button klickt.
